Política de Privacidad

Última actualización: 3 de enero de 2026

La presente Política de Privacidad describe cómo la aplicación móvil Pickly AI (en adelante, la “Aplicación”) trata la información del usuario. El uso de la Aplicación implica la aceptación de esta Política.

1. INFORMACIÓN QUE RECOPILAMOS

1.1. Datos personales

Pickly AI no recopila, almacena ni procesa datos personales identificativos del usuario. La Aplicación no requiere registro, inicio de sesión ni creación de cuentas.

1.2. Imágenes

Para el funcionamiento principal de la Aplicación, el usuario selecciona entre 2 y 5 imágenes desde la galería de su dispositivo. El tratamiento se realiza así:

  • Las imágenes seleccionadas se envían de forma temporal y cifrada a nuestro servidor.
  • El servidor procesa las imágenes mediante un modelo de IA con el único fin de generar una puntuación.
  • Una vez devuelto el resultado:
    • Las imágenes se eliminan automáticamente e inmediatamente del servidor.
    • No se almacenan ni conservan copias.

1.3. Permisos del dispositivo

  • Acceso a la galería de fotos, exclusivamente para la selección de imágenes.
  • Permiso de notificaciones, solo si el usuario decide habilitarlas.

2. FINALIDAD DEL TRATAMIENTO

El tratamiento se limita a procesar las imágenes para generar la puntuación. Las imágenes no se utilizan para entrenar modelos de IA, mejorar algoritmos ni para fines comerciales.

3. BASE LEGAL

El tratamiento se fundamenta en el consentimiento del usuario al utilizar la Aplicación y en la ejecución del servicio solicitado.

4. CUSTODIA Y COMUNICACIÓN

Pickly AI no cede, comparte ni vende imágenes a terceros. Las fotografías se envían únicamente a nuestro servidor controlado.

5. PROCESAMIENTO DE IMÁGENES

Tiempo de conservación: 0 minutos.

Las imágenes se eliminan de forma irreversible inmediatamente tras completarse el análisis.

6. SEGURIDAD

Aplicamos transmisión cifrada HTTPS/SSL y eliminación automática para garantizar que no exista almacenamiento permanente.

7. DERECHOS DEL USUARIO

Dado que no almacenamos datos personales ni conservamos copias de las imágenes, el usuario mantiene el control absoluto eliminando la App o revocando permisos.
